# 🌦️ ClimaNova

ClimaNova es una aplicación web desarrollada con React y TypeScript que permite consultar el clima actual de cualquier ciudad utilizando la API de Open-Meteo.

La aplicación obtiene las coordenadas de la ciudad mediante la Geocoding API y luego consulta la temperatura actual usando la Weather Forecast API.

---

# 🚀 Funcionalidades

- Buscar clima por ciudad
- Temperatura en tiempo real
- Manejo de errores
- Validaciones de entrada
- Uso de async/await
- Interfaz moderna y responsive
- Consumo de APIs con Fetch

---

⚙️ Cómo Ejecutar el Proyecto
1. Crear el proyecto con Vite
npm create vite@latest

Seleccionar:

Framework: React
Variant: TypeScript

2. Entrar a la carpeta del proyecto
cd ClimaNova

3. Instalar dependencias
npm install

4. Ejecutar el servidor de desarrollo
npm run dev

5. Abrir en el navegador
http://localhost:5173

La aplicación se ejecutará utilizando Vite y permitirá consultar el clima en tiempo real de diferentes ciudades.

-🌐 APIs Utilizadas
Geocoding API

Se utiliza para obtener las coordenadas de la ciudad ingresada por el usuario.

Weather Forecast API

Se utiliza para consultar la temperatura actual y la velocidad del viento.
